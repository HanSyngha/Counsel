# Counsel - 위인의 목소리

Get wisdom from history's greatest minds through AI-powered conversations.

## About

Counsel connects you with philosophers, leaders, and thinkers throughout history. Ask questions about life, decisions, and challenges, and receive thoughtful advice inspired by their wisdom and teachings.

## Features

- **19 Historical Personas**: From Socrates and Buddha to Steve Jobs and Gandhi
- **16 Languages**: Korean, English, Japanese, Chinese, Spanish, French, German, Portuguese, Russian, Arabic, Hindi, Thai, Vietnamese, Indonesian, Malay, Turkish
- **AI-Powered Advice**: Personalized responses using Grok API
- **Citation System**: Each response includes relevant historical quotes and sources
- **Save & Share**: Bookmark favorite advice and share wisdom with others
- **Dark Theme**: Elegant dark UI designed for focused reading

## Personas

### Philosophy
- Socrates, Plato, Aristotle, Seneca, Confucius, Laozi, Rumi

### Religion
- Jesus, Buddha, Muhammad, Krishna, Brahma

### History
- Abraham Lincoln, Napoleon, Steve Jobs, Mahatma Gandhi

### Literature
- Sherlock Holmes, Albus Dumbledore, Leo Tolstoy

## Tech Stack

- **Framework**: Flutter 3.x
- **State Management**: Riverpod
- **Local Storage**: Hive
- **AI Backend**: Grok API (xAI)
- **Ads**: Google AdMob

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/HanSyngha/Counsel.git
cd Counsel
```

2. Create `.env` file with your API key
```
GROK_API_KEY=your_api_key_here
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```

## Build

```bash
# APK
flutter build apk --release

# App Bundle
flutter build appbundle --release
```
